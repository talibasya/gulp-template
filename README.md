# GULP template

using:
```
{
    "gulp": "^3.9.1",
    "gulp-cache": "^0.4.5",
    "gulp-concat": "^2.6.0",
    "gulp-imagemin": "^3.0.1",
    "gulp-rename": "^1.2.2",
    "gulp-sass": "^2.3.2",
    "gulp-uglify": "^1.5.4"
  }
```

Install npm dependencies:
```
npm i
```

Run and watch:
```
npm start
```

Other commands:
- `npm run scripts`: compiles `*.js` files in 'src/js'
- `npm run sass`: compiles `*.scss` files in 'src/scss'
- `npm run images`: resizes images in 'src/images'
